Restaiuranteboard

Plataforma web para gestionar un Restaurante Virtual con login por roles, menú, pagos, cocina y delivery.

Iniciar sesión Crear cuenta

Accesos por rol

Login para cliente, administrador, cajero, cocinero y repartidor con validaciones de seguridad y recuperación de clave.

Flujos con correo

Confirmación por código en registro, activación de personal y recuperación de contraseña usando SMTP configurado.

Venta y operación

Menú para clientes, validación de pagos en caja, avance de órdenes en cocina y seguimiento para entrega.

Gestión del negocio

Setup inicial con nombre, logo, términos y medios de pago; además de administración de ingredientes y productos.

Login y cuentas

Inicio de sesión robusto, registro con validaciones completas, términos y confirmación por código para cada tipo de usuario.

Setup y administración

Configuración SMTP, información del negocio, medios de pago, y creación de ingredientes/productos para el catálogo.

Experiencia cliente

Navegación de menú, recomendaciones por modelo de IA, carrito, checkout, trazabilidad, calificación de servicio y edición de datos.

Operación interna

Validación de pagos, gestión de cocina y delivery, dashboard y proyecciones con IA para decisiones de abastecimiento.
